# CPSC 548 Syllabus

## Course Description

Name: Software Maintenance

Units: 3

Summary: The Student will learn the principles of generating maintainable software. The course will also cover Theory and practice of maintaining large scale software and application of maintenance metrics. In this course, we will cover the problems in maintaining software systems, building software in view of the maintenance problems, the activity of maintenance and some management issues in maintenance.

Prerequisites: CPSC 362 or equivalent work experience.
